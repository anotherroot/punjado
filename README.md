# punjado
A TUI/CLI tool for copying a handful of files of a project in a AI friendly format.

## Features
- [ ] git ignore filter
- [ ] select only part of file 
- [ ] selection history
- [ ] save various selections
- [ ] global data file
- [ ] save setting profiles 
- [ ] select based on regex 
- [ ] ignore list
- [ ] config file
- [ ] customizable keymaps
